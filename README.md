# MediFamba

# (Work In Progress)


## Hospital Appointment Management System

MediFamba is a web application designed to simplify hospital appointment management. It supports multiple user roles—patients, doctors, and administrators—ensuring that each user interacts with the system according to their permissions.

## Features

- Role-Based Dashboards: Redirect users to different dashboards based on their role.

- User Management: Admins can manage doctors and patients.

- Appointment Scheduling: Patients can book appointments with available doctors.

- Doctor Workflow: Doctors can view, confirm, reschedule, or cancel appointments.

## Technologies
- Python & Django (backend)
- HTML, CSS, JavaScript (frontend)
- Bootstrap (frontend)
